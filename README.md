## GitHub Pages 활성화 방법
1. 저장소의 Settings > Pages로 이동
2. Source를 `main`(또는 사용 중인 기본 브랜치)와 `/ (root)`로 설정
3. Save를 클릭한 후, 제공되는 URL(예: https://xownstj05-rgb.github.io/homekma123/)에 `BABY.html`을 붙여 접속

※ BABY.html 파일은 CDN(예: reveal.js, Chart.js, Google Fonts)을 사용하므로, Raw URL이나 GitHub Pages에서 정상적으로 로드됩니다.

## 로컬에서 보기
1. BABY.html 파일을 다운로드
2. 브라우저로 파일을 열거나(파일 > 열기), 간단한 로컬 서버(예: `python -m http.server`)로 서빙하여 확인하세요.
